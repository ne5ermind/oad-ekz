# Данные как основа анализа. Методы сбора данных. Типы данных. Примеры данных на транспорте

## 1. Методы сбора данных

### Ручной сбор
Процесс сбора данных без использования автоматизированных инструментов из цифровых или физических носителей.

| Плюсы | Минусы |
| :--- | :--- |
| Возможность использования любых источников | Требуется обучение персонала |
| Сбор данных, недоступных для роботов | Большие временные затраты |
| | Высокий риск ошибок (человеческий фактор) |

### Автоматический мониторинг цифровых источников
Наблюдение за онлайн-ресурсами, базами данных и электронными документами.

| Плюсы | Минусы |
| :--- | :--- |
| Обработка больших массивов данных (Big Data) | Ограничение только онлайн-ресурсами |
| Экономия времени и режим реального времени | Высокие затраты на разработку ПО |
| Исключение человеческого фактора | |

### Сбор данных с датчиков (IoT)
Мониторинг физических параметров оборудования и окружающей среды в реальном времени.

| Плюсы | Минусы |
| :--- | :--- |
| Высокая скорость получения данных | Затраты на разработку и оборудование |
| Работа с огромными объемами информации | Необходимость тех. обслуживания датчиков |
| Экономия временных ресурсов | Требуется обучение персонала |

---

## 2. Типы данных

*   **Метаданные** — «данные о данных». Описывают структуру, формат, характеристики и контекст основной информации.
*   **Справочные данные** — стандартизированные и стабильные данные (классификаторы, справочники). Обеспечивают совместимость и правильную категоризацию.
*   **Основные данные (Master Data)** — данные о ключевых бизнес-сущностях организации: клиентах, продуктах, сотрудниках, технологиях и материалах.

---

## 3. Примеры данных на транспорте

На железнодорожном транспорте для планирования и контроля используются следующие показатели:

1.  **Объем перевозок:** количество перевезенных тонн груза.
2.  **Показатели использования подвижного состава:** 
    *   Оборот вагона;
    *   Коэффициент порожнего пробега;
    *   Время простоя.
3.  **Статистика безопасности:** количество отказов технических средств, инцидентов, задержек поездов.
4.  **Пассажиропоток:** количество пассажиров на маршрутах, анализ сезонных колебаний.
5.  **Технические данные:** 
    *   Показания датчиков температуры букс;
    *   Уровень топлива в локомотивах;
    *   Геометрические параметры состояния пути.
