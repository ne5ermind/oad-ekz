# 8 методов анализа данных, применяемых на железнодорожном транспорте. Анализ Парето: вид анализа, назначение, алгоритм реализации

## 1. 8 ключевых методов анализа данных
В практике управления качеством и анализа процессов на железнодорожном транспорте применяются следующие методы:

1.  **Анализ Парето** — расстановка приоритетов.
2.  **Диаграмма Исикавы** — поиск причинно-следственных связей («рыбий скелет»).
3.  **Корреляционный анализ (Диаграмма рассеяния)** — выявление зависимостей между факторами.
4.  **Исследование разброса параметров (Гистограммы)** — анализ распределения данных.
5.  **Z-график и исследование вариабельности** — контроль стабильности процессов.
6.  **Формат корректирующих действий** — стандартизация устранения отклонений.
7.  **Метод «5W + 1H + 1S»** — структурированное описание проблемы (Who, What, Where, When, Why + How + Safety).
8.  **Методы решения проблем (8 шагов)** — комплексный алгоритм выхода из проблемных ситуаций.

## 2. Анализ Парето

**Анализ Парето** — это метод расстановки приоритетов, основанный на эмпирическом правиле: *«20% усилий дают 80% результата»*.

*   **Вид анализа:** статистический, причинно-следственный, метод приоритетов.
*   **Назначение:** определение ключевых факторов («жизненно важного меньшинства»), которые вызывают основную долю проблем (80% брака, задержек поездов или жалоб).

### Алгоритм реализации

#### Этап 1. Подготовка входных данных
*   Сбор первичной информации по видам нарушений или отказов.
*   Разработка и заполнение сводной таблицы.
*   **Состав таблицы:** типы случаев, количество появлений (частота), накопленная сумма и процентная доля каждого типа в общем объеме.

#### Этап 2. Построение осей
Для построения диаграммы необходимо подготовить три оси:
*   **Горизонтальная ось:** интервалы по числу типов (категорий), упорядоченные по убыванию частоты.
*   **Левая вертикальная ось:** шкала от 0 до общей суммы случаев (абсолютные значения).
*   **Правая вертикальная ось:** шкала от 0 до 100% (относительные значения).

#### Этап 3. Построение диаграммы
1.  **Столбиковая диаграмма:** отображает частоту возникновения каждого типа событий (высота столбца соответствует значению на левой оси).
2.  **Кумулятивная кривая (Кривая Лоренца):** наносятся точки накопленных сумм в процентах, которые последовательно соединяются отрезками.

#### Этап 4. Оформление и обозначения
Для корректной интерпретации на диаграмме должны быть размещены:
*   Название диаграммы.
*   Четкая разметка всех осей.
*   Описание объекта исследования.
*   Указание периода сбора информации.
