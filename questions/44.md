# Алгоритм кластеризации DBScan

**Алгоритм DBSCAN** — кластеризация на основе **плотности**, автоматически находит кластеры произвольной формы и **выделяет шум** (выбросы). Не требует задания числа кластеров!

## **Ключевые термины**

```
ε (эпсилон) — максимальное расстояние между точками в кластере
minPts — минимальное число точек для "плотной области" (обычно 3-10)

3 типа точек:
• КОРЕВАЯ (Core) — ≥ minPts соседей в радиусе ε
• ГРАНИЧНАЯ (Border) — < minPts соседей, но рядом с корневой
• ШУМОВАЯ (Noise) — ни то, ни другое
```


## **Алгоритм**

```
1. Берём непосещённую точку
2. Если КОРЕВАЯ → создаём кластер
3. Добавляем ВСЕ точки в радиусе ε от корневых
4. Расширяем кластер рекурсивно (цепная реакция)
5. Повторяем для всех точек
```


##  **Пример для экзамена: магазины в городе**

```
Данные: координаты (X,Y) 9 магазинов
ε = 2 км, minPts = 3

Точки:
A(1,1) B(1,2) C(1,3)    D(5,5) E(5,6)    F(10,10)
G(1,1.5)                H(9,10)           I(11,11)
```

**Итерация 1:**

```
Проверяем A(1,1):
• Соседи в ε=2: B,C,G (3 точки ≥ minPts=3)
• A = КОРЕВАЯ → Кластер 1

Расширяем от A:
• B(1,2): соседи A,C,G → КОРЕВАЯ
• C(1,3): соседи A,B,G → КОРЕВАЯ  
• G(1,1.5): соседи A,B,C → КОРЕВАЯ

 Кластер 1: A,B,C,G (4 магазина в центре)
```

**Итерация 2:**

```
D(5,5): соседи E (1 < 3) → пока ШУМ
E(5,6): соседи D (1 < 3) → пока ШУМ

F(10,10): соседей нет → ШУМ
H(9,10): сосед F (1 < 3) → ШУМ  
I(11,11): сосед H (1 < 3) → ШУМ
```

**Итерация 3:**

```
E(5,6): соседи D (1 < 3) → ГРАНИЧНАЯ (примыкает к D)
D = КОРЕВАЯ → Кластер 2: D,E

 ИТОГО:
Кластер 1: A,B,C,G (центр города)
Кластер 2: D,E (пригород)  
Шум: F,H,I (одиночки)
```


##  **DBSCAN & K-means**

| Аспект | DBSCAN | K-means |
| :-- | :-- | :-- |
| **Число кластеров** | Автоматически | Задаём k |
| **Форма кластеров** | Любая | Сферические |
| **Шум** | Выделяет | Включает в кластеры |
| **Параметры** | ε, minPts | k |
| **Скорость** | Средняя | Быстрая |



```
"DBSCAN = плотностная кластеризация:
1. ε-окрестность каждой точки
2. Коревые точки (≥minPts соседей)
3. Расширение кластера от корневых
4. Шум = точки вне плотных областей"
```
