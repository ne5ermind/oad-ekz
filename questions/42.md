# Визуализация метрических данных. Корреляционная матрица

Метрические данные (количественные, непрерывные/дискретные числа вроде роста, веса, дохода) визуализируют для показа распределения, трендов и связей.

## Визуализация метрических данных

**Основные графики:**

- **Гистограмма** — распределение (частота интервалов роста)
- **Box-plot** — квартили, медиана, выбросы
- **Линейный график** — динамика во времени (продажи по дням)
- **Диаграмма рассеивания** — связь между двумя метриками
- **Корреляционная матрица** — взаимосвязи всех пар метрик

## Корреляционная матрица

Таблица/тепловая карта, где каждая ячейка показывает **корреляцию** (от -1 до +1) между двумя метриками.
