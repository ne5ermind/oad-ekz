# Неметрические и метрические данные. Типы переменных. Примеры данных. Отображение типов данных в pandas

## 1. Типы переменных в данных

В статистике и машинном обучении переменные делятся на две основные группы:

### ➤ Категориальные (Categorical) — неметрические
- Описывают **классы или категории**.
- Не имеют числового значения в прямом смысле.
- Подразделяются на:
  - **Номинальные** — без порядка  
    *Примеры:* пол, профессия, цвет глаз
  - **Порядковые** — с порядком  
    *Примеры:* оценки, уровень удовлетворённости

### ➤ Числовые (Numerical) — метрические
- Описывают **величину, количество, размер**.
- Можно измерять, складывать, вычитать, делить.
- Подразделяются на:
  - **Дискретные** — целые числа  
    *Примеры:* количество покупок, возраст, число детей
  - **Непрерывные** — любое число в диапазоне  
    *Примеры:* доход, вес, время в пути

---

## 2. Соотношение типов переменных и шкал измерения

| Тип переменной       | Соответствующая шкала      | Примеры                          |
|----------------------|----------------------------|----------------------------------|
| Номинальная          | Номинальная шкала          | Пол, профессия, цвет глаз        |
| Порядковая           | Порядковая шкала           | Оценки, уровень удовлетворённости |
| Дискретная           | Интервальная / Шкала отношений | Количество покупок, возраст     |
| Непрерывная          | Шкала отношений            | Доход, вес, время                |

>  В `pandas` мы не видим напрямую «шкалу измерения», но можем определить тип переменной — и по нему понять, какие операции допустимы.

## 3. Примеры данных по типам

###  Номинальные переменные (категориальные без порядка)

#### Пример: пол, страна, марка автомобиля
```
data = {
'Пол': ['М', 'Ж', 'Ж', 'М'],
'Страна': ['Россия', 'США', 'Германия', 'Россия'],
'Бренд': ['Apple', 'Samsung', 'Huawei', 'Apple']
}
```
