## 1. Понятие группировки данных
Группировка - это расчленение совокупности данных на группы по определенному существенному признаку. Является одним из методов описательной статистики

#### Цели группировки:
*   Упрощение воспринятия большого массива данных
*   Сравнение характеристик различных сегментов
*   Выявление взаимосвязей между признаками

---

## 2. Общий алгоритм группировки
В анализе данных классический алгоритм состоит из 3 этапов:

#### 1. Разделение
Данные разбиваются на части на основе выбранного ключа (столбца или условия)
#### 2. Применение 
К каждой группе применяется какая либо агрегирующая функция
#### 3. Комбинирование
Результат вычислений объединяется в новую структуру данных

---

## 3. Реализация в Pandas
В библиотеке Pandas основным инструментом является метод `groupby()`

#### Базовый синтаксис:

```python
import pandas as pd

# Создание dataframe
df = pd.DataFrame({
    'category': ['a', 'b', 'c', 'd', 'e'],
    'values': [19, 84, 69, 42, 0],
    'names': ['stepa', 'kirill', 'misha', 'grisha', 'alexey borisov']
})

# Группировка по одному столбцу и расчет среднего
grouped = df.groupby('category')['values'].mean()
```

---

## 4. Агрегирующие функции

### A) Для числовых значений
*   `.sum()` - сумма
*   `.mean()` - среднее арифмитическое
*   `.median()` — медиана
*   `.std()` / `.var()` — стандартное отклонение и дисперсия
*   `.min()` / `.max()` — поиск экстремумов
*   `.count()` — количество записей (включая NaN)
*   `.size()` — размер группы (включая NaN)
### Б) Для категориальных и текстовых данных
*   `.unique()` — количество уникальных значений в группе
*   `.first()` / `.last()` — получение первого или последнего значения
*   `.describe()` — вывод полной описательной статистики по каждой группе
### В) Универсальный метод `.agg()`
Позволяет применять сразу несколько функций или разные функции к нескольким столбцам:

```python
res = df.groupby('category').agg({
    'values': ['sum', 'mean'],
    'names': 'count'
})
```
