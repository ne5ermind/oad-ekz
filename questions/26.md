# ETL\ELT. Этапы, плюсы и минусы. Примеры


### Это процессы управления данными, главное отличие между ними - порядок выполнения операций.

---

## ETL 

#### ETL - Extract -> Transform -> Load

#### ETL - Извлечение -> Трансформация и очистка -> Выгрузка в хранилище

#### Плюсы:

* Легко разрабатывать
* Сразу очищаем и кладём на витрину

#### Минусы:

* При большом объёме данных очень медленная работа
* Ресурсоёмкие операции

#### Примеры: Microsoft SQL Server

---



## ELT

#### ETL - Extract -> Load -> Transform

#### ETL - Извлечение -> Выгрузка в хранилище -> Трансформация и очистка

#### Плюсы:

* Максимальная скорость обработки
* Возможность обработки большого количетва данных

#### Минусы:

* Загружается много лишнего во временное хранилище
