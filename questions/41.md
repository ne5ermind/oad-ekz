
# Визуализация дискретных данных. Разница между Столбчатой диаграммой и Гистограммой. Диаграмма рассеивания.

Визуализация дискретных данных использует графики для отображения счётных величин (количество событий). Столбчатая диаграмма подходит для категорий, диаграмма рассеяния — для связей между переменными.

## Визуализация дискретных данных

Дискретные данные (0, 1, 2 детей; количество продаж) визуализируют:

- **Столбчатая диаграмма** — для категорий (кол-во М/Ж, возрастные группы)
- **Гистограмма** — редко (только если много соседних целых значений)
- **Круговая диаграмма** — доли категорий (≤5 групп)
- **Тепловая карта** — частоты в таблице


## Разница между Столбчатой диаграммой и Гистограммой

| Аспект | Столбчатая диаграмма | Гистограмма |
| :-- | :-- | :-- |
| **Данные** | Категории (М/Ж, регионы) | Непрерывные (рост 160-170 см) |
| **Оси** | X — категории, Y — частота | X — интервалы, Y — плотность |
| **Столбцы** | **С пробелами** между ними | **Соприкасаются** (нет пробелов) |
| **Пример** | Продажи: Москва=100, СПб=80 | Рост: 160-170=5 чел., 170-180=12 |

**Ключевое отличие для:** "Столбцы **касаются** = гистограмма (непрерывные), **с пробелами** = столбчатая (категории/дискретные)"

## Диаграмма рассеивания (Scatter Plot)

Показывает **связь между двумя переменными** — точки по осям X и Y.

- **Для дискретных**: одна или обе оси дискретные
- **Пример**: X=возрастная группа (18-25, 26-35), Y=доход группы

**Когда использовать:**

- Столбчатая: сравнение категорий по высоте
- Гистограмма: форма распределения непрерывных
- Рассеивания: корреляция (положительная/отрицательная наклон)
