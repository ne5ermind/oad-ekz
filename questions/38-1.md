# **Описательная статистика. Инструменты для непрерывных даных. Примеры.**

Описательная статистика для непрерывных данных использует меры центра, разброса и визуализацию, чтобы суммировать измеряемые величины вроде роста или времени.

## Основные меры центра (ИНСТРУМЕНТЫ)

### Для непрерывных данных (рост, вес, температура) применяют:

   * Среднее арифметическое: сумма значений / количество — типичное значение, но чувствительно к выбросам.

   * Медиана: срединное значение в отсортированном списке — устойчиво к аномалиям.

   * Мода: наиболее частое значение (если есть пики в распределении)

### Меры разброса

   * Размах: максимум - минимум — простая оценка spread (Spread (разброс) — это степень, насколько данные различаются между собой, насколько они "разбросаны" вокруг среднего значения.).

   * Дисперсия: средний квадрат отклонений от среднего (σ2=∑(xi−xˉ)2nσ2=n∑(xi−xˉ)2).

   * Стандартное отклонение: σ=σ2σ=σ2
    — разброс в единицах данных (68% в xˉ±σxˉ±σ для нормального распределения).

   * Квартили (Q1, Q3): 25% и 75% данных — для box-plot.

### Визуализация

   * Гистограмма: столбцы без пробелов показывают форму распределения (колоколообразное, скошенное).

   * Box-plot (ящик с усами): медиана, квартили, выбросы — выявляет асимметрию и outliers.

   * Q-Q plot: проверка на нормальность (точки вдоль линии — нормально).

## Примеры

Рост студентов (см): [160, 165, 170, 172, 175, 180, 190] 

   * Среднее: 173 см

   * Медиана: 172 см

   * Дисперсия: ~95 (σ ≈ 9.7 см)

   * Гистограмма: пик около 170 см

   * Box-plot: усами 160-180, выброс 190 см

Время доставки (мин): [25.3, 28.1, 30.4, 32.7, 45.2]

   * Среднее: 32.3 мин (выброс 45.2 тянет вверх)

   * Медиана: 30.4 мин (лучше отражает типичное)

   * Станд.откл.: ~7.5 мин — большинство 25-40 мин
