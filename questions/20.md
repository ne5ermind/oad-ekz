# Классификация методов анализа данных по фунициональному назначению. 4 уровня аналитики по GARTNER. Примеры инструментов и методов
Классификация методов анализа данных по их функциональному назначению позволяет понять, какие задачи решает аналитика. Наиболее признанной моделью в этой области является
шкала зрелости аналитики Gartner

## 4 уровня аналитики по Garter
Модель выстроена по приципу возрастания сложности и ценности для бизнеса

#### 1. Дескриптивная аналитика - "Что произошло?"
Базовый уровень, на котором данные собираются и визуализируются для понимания текущей ситуации  или итогов периода 
*   Цель - констатация фактов
*   Методы - группировка, аггрегирование, расчет частот, построение отчетов
*   Инструменты - excel, отчеты, простые дашборды
*   Пример - отчет о продажах за прошлый месяц

#### 2. Диагностическая аналитика - "Почему это произошло?"
На этом уровне присходит поиск причинно-следственных связей и выявленние факторов, влияющих на результат 
*   Цель - поиск аномалий и зависимостей 
*   Методы - детализация, корреляционный анализ, факторный анализ, сравнение выборок
*   Инструменты - продвинутые BI-системы, статистические пакеты
*   Пример - выяснение того, что продажи упали из-за логистического сбоя

#### 3. Предиктивная аналитика - "Что произойдет?"
Использует исторические данные для построения моделей, которые предсказывают будущие события
*   Цель - прогнозирование вероятностей
*   Методы - регрессионный анализ, анализ времянных рядов, машшине обучение
*   Инструменты - Python (Scikit-learn, PyTorch, XGBoost), R, облачные платформы (AWS, Google Vertex AI)
*   Пример - прогноз оттока клиентов в следующем квартале из-за падения метеорита

#### 4. Прескриптивная аналитика - "Что нужно сделать, чтобы это произошло?"
Высший уровень аналитики. Предлагает конкретные действия для достижения поставленных целей
*   Цель - оптимизация и автоматизация принятия решений
*   Методы - математическое программирование, симуляционное моделирование, обучение с подкреплением (Reinforcement Learning)
*   Инструменты - специализированное ПО (AnyLogic), системы на базе ИИ-агентов
